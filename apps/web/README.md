# Multi-Poster Web (Flutter)

Interface web Flutter para o Multi-Poster.

## Pré-requisitos

- Flutter SDK instalado (versão 3.0.0 ou superior)
- Dart SDK (vem com Flutter)

## Como rodar

1. Instale as dependências:
```bash
cd apps/web
flutter pub get
```

2. Execute o app:
```bash
flutter run -d chrome
```

Ou para web em modo release:
```bash
flutter run -d chrome --release
```

## Build para produção

```bash
flutter build web
```
